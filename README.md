bash scripts
------------
For make.sh and testing.sh, I recommend using a ~/bin directory and then making
a symbolic link to both scripts.

    ln -s <path-to-file>/make.sh ~/bin/make
    ln -s <path-to-file>/testing.sh ~/bin/testing
